# Portfolio - Chahinez Ayadi

Un portfolio moderne et fluide avec un design informatique élégant, créé pour une étudiante en informatique.

## 🎨 Caractéristiques

- **Design moderne** : Interface épurée avec un style informatique/tech
- **Animations fluides** : Transitions douces et effets interactifs
- **Responsive** : Adapté à tous les écrans (mobile, tablette, desktop)
- **Navigation fluide** : Scroll smooth et menu interactif
- **Couleurs simples** : Palette sobre avec des accents colorés

## 📋 Sections

1. **Accueil** : Hero section avec effet de typing
2. **Profil** : Présentation personnelle
3. **Compétences** : Technologies maîtrisées (Front-end, Back-end, Outils)
4. **Parcours** : Formation et études avec timeline
5. **Entreprise** : Expériences professionnelles
6. **Projets** : Portfolio de projets réalisés
7. **Réalisations** : Accomplissements
8. **Veille** : Veille technologique
9. **Contact** : Formulaire de contact

## 🚀 Installation

1. Clonez ou téléchargez le projet
2. Ouvrez `index.html` dans votre navigateur
3. C'est tout ! Aucune dépendance requise

## 📝 Personnalisation

### Modifier les informations personnelles

1. **Nom et description** : Modifiez dans `index.html` les sections correspondantes
2. **Compétences** : Ajoutez/modifiez les tags dans la section compétences
3. **Projets** : Personnalisez les cartes de projets avec vos propres projets
4. **Contact** : Mettez à jour les liens email, LinkedIn, GitHub dans la section contact

### Modifier les couleurs

Dans `styles.css`, modifiez les variables CSS dans `:root` :

```css
:root {
    --primary-color: #6366f1;    /* Couleur principale */
    --secondary-color: #8b5cf6;  /* Couleur secondaire */
    --bg-dark: #0a0a0f;          /* Fond sombre */
    /* ... */
}
```

### Ajouter votre photo

Remplacez la section `.placeholder-image` dans la section profil par votre image :

```html
<img src="votre-photo.jpg" alt="Photo de profil" class="profil-photo">
```

## 🛠️ Technologies utilisées

## 📱 Responsive

Le site est entièrement responsive et s'adapte automatiquement à :
- Mobile (< 640px)
- Tablette (640px - 968px)
- Desktop (> 968px)

## 🎯 Fonctionnalités JavaScript

- Navigation active selon la section visible
- Menu hamburger pour mobile
- Effet de typing sur le nom
- Animations au scroll
- Formulaire de contact (à connecter à un backend)
- Effets de parallaxe
- Effets hover sur les cartes
